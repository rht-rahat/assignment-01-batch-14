# 📨 AI Prompt: Newsletter Component

This document contains the specifications and requirements for generating a clean, modern newsletter section using vanilla web technologies.

---

## 📋 Requirements Specification

### 1. Structure (HTML)
* **Component Type:** Simple & clean Newsletter section.
* **Elements Needed:**
    * `Title` (Main heading)
    * `Subtitle` (Supporting description)
    * `Input Field` (For email capture)
    * `Subscribe Button` (Action element)

### 2. Styling (Raw CSS)
* **Methodology:** Pure, raw CSS (No external frameworks or preprocessors).
* **Layout:** Modern, inline, or block-focused layout that flows naturally.
* **Color Palette:** * **Primary Action Button:** `#2563eb` (Blue Color)
* **Typography & Accents:** Clean, readable hierarchy with complementary tones.
